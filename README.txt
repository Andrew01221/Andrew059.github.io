Bubbly CV Website (HTML + CSS)

Files:
- index.html
- styles.css
- assets/profile.jpg
- assets/AndrewScutt_CV.pdf

How to run locally:
1) Open index.html by double-clicking it
   OR
2) Run a local server (recommended so links behave the same as on hosting):
   - Python:
     python -m http.server 8000
   Then visit: http://localhost:8000

How to customize:
- Replace your photo: assets/profile.jpg
- Replace your CV: assets/AndrewScutt_CV.pdf (keep the same name)
- Edit About/Highlights text in index.html

Deploy (simple options):
- GitHub Pages: push these files to a repo, enable Pages, select main branch /root.
- Netlify/Vercel: drag-and-drop the folder or connect the repo.
